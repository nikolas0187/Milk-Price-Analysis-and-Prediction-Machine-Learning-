## Project Overview
This project was developed during an Artificial Intelligence bootcamp and focuses on data cleaning, integration, exploratory analysis, and price prediction using historical milk price data from 2023 to 2025 in Colombia.

## Workflow
- Data cleaning for each year using Python
- Integration of multiple datasets into a unified database
- Feature engineering and preprocessing
- Price prediction using machine learning techniques

## Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## Outcome
The project delivers a clean and structured dataset and a basic predictive model to support trend analysis and data-driven decision-making.
